# Aryan Portfolio Website

Modern responsive frontend developer portfolio website built using HTML, Tailwind CSS and JavaScript.

## Features
- Responsive modern UI
- Dark themed design
- Animated hero section
- Skills & About section
- CRUD-based BGMI Tournament Manager
- LocalStorage support
- Mobile friendly layout

## Technologies Used
- HTML5
- Tailwind CSS
- JavaScript

## CRUD Functionality
The BGMI Tournament Manager supports:
- Create Team
- Read Team List
- Update Team Name
- Delete Team
